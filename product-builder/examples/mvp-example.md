```markdown
# MVP — FreelaTask

## Problema
Freelancers brasileiros perdem o controle financeiro dos projetos quando trabalham com múltiplos clientes. Usam WhatsApp + planilhas que não se integram, e frequentemente esquecem de cobrar ou cobram valores errados.

## Público-Alvo
**Usuário primário:** João, 28, desenvolvedor freelancer, 5-8 clientes simultâneos, fatura R$ 6k-12k/mês, usa planilha Google + WhatsApp hoje.

## Proposta de Valor
Primeira ferramenta de gestão de projetos construída especificamente para freelancers brasileiros, com foco em clareza financeira no formato BRL.

## Features do MVP

| # | Feature | Motivação |
|---|---------|-----------|
| 1 | Criar tarefa (título + cliente + valor + status) | Core do produto |
| 2 | Listar tarefas com filtro por status | Visualizar trabalho atual |
| 3 | Alterar status da tarefa | Marcar progresso |
| 4 | Painel financeiro (total, recebido, a receber) | Clareza financeira |
| 5 | Excluir tarefa | Gestão básica |

## Fora do MVP
- Autenticação/login (primeira versão offline/local)
- App mobile
- Integração com NF
- Relatórios avançados
- Múltiplos usuários

## Jornadas Principais

### Jornada 1: Cadastrar novo projeto
1. Clica "Nova Tarefa"
2. Preenche título, cliente, valor, prioridade
3. Salva
4. Vê o card na lista com valor em BRL

### Jornada 2: Monitorar financeiro
1. Abre o app
2. Vê painel: Total / Recebido / A receber
3. Marca tarefa como "concluído" quando pago
4. Vê valor se mover para "Recebido"
```
